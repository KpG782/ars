📋 To Create Your Icon:

  1. Choose one of the designs above (or create your own)
  2. Use any design tool:
    • Canva (free, easy) - canva.com
    • Figma (free, professional) - figma.com
    • Adobe Express - adobe.com/express
  3. Create a 1024x1024px image
  4. Use the teal color: #00BFA5
  5. Make "ARS" text large and bold
  6. Export as PNG
  7. Save as app_icon.png in the assets/icon/ folder

🎨 Design Tips:

  • Keep it simple - icons are viewed at small sizes
  • Use high contrast (white text on teal, or teal on white)
  • Avoid thin lines - they disappear at small sizes
  • Center the main elements
  • Test at 48x48px to see how it looks on actual devices

⚡ Quick Generate (After adding icon):

Run in terminal:

flutter pub get
flutter pub run flutter_launcher_icons
flutter clean
